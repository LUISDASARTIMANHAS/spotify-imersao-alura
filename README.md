# Projeto Clone da Interface do Spotify

## 1. Resumo

Este projeto acadêmico, desenvolvido durante o programa Imersão Front-End da Alura, consiste na recriação da interface de usuário (UI) da plataforma de streaming de música Spotify. O objetivo principal foi aplicar e aprofundar conhecimentos em tecnologias web modernas, focando na construção de layouts responsivos, componentização e manipulação de dados estáticos para simular uma aplicação dinâmica.

## 2. Proposta e Objetivo

O propósito central do projeto é educacional, servindo como um estudo de caso prático para o desenvolvimento de uma interface web complexa e amplamente reconhecida. Os objetivos específicos incluem:

- **Estruturação Semântica:** Utilizar HTML5 para construir uma estrutura de página clara e acessível.
- **Estilização Avançada:** Empregar CSS3, incluindo Flexbox e Grid Layout, para replicar o design visual do Spotify, garantindo fidelidade ao layout original.
- **Responsividade:** Adaptar a interface para diferentes tamanhos de tela, como desktops, tablets e dispositivos móveis, utilizando Media Queries.
- **Componentização:** Modularizar a interface em componentes reutilizáveis, uma prática explorada tanto na versão com JavaScript puro quanto na versão com React.
- **Simulação de API:** Utilizar um arquivo JSON local para carregar e exibir dinamicamente uma lista de artistas, simulando o consumo de dados de uma API externa.

## 3. Tecnologias Utilizadas

O projeto foi desenvolvido em duas versões para explorar diferentes abordagens do desenvolvimento front-end:

### Versão Principal (Vanilla)
- **HTML5:** Para a marcação e estrutura do conteúdo.
- **CSS3:** Para estilização, layout e responsividade.
- **JavaScript (ES6+):** Para manipulação do DOM e interatividade.

### Versão Secundária (React)
- **React:** Biblioteca para construção de interfaces de usuário baseadas em componentes.
- **JSX:** Extensão de sintaxe para JavaScript, utilizada para descrever a aparência da UI.
- **Node.js / npm:** Para gerenciamento de pacotes e execução do ambiente de desenvolvimento.

## 4. Estrutura do Projeto

O repositório está organizado da seguinte forma:

- **/src/**: Contém os arquivos da aplicação principal (HTML, CSS, JS e assets).
- **/spotify-react/**: Contém uma implementação alternativa da interface utilizando a biblioteca React.
- **index.html**: Ponto de entrada da aplicação principal.
- **README.md**: Esta documentação.
